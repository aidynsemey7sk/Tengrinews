# Tengrinews
Интернет газета 

#### Ссылка на сайт https://tengri-frontend.herokuapp.com/
 - django проект для газеты
 
 ## Project setup
```
pip3 install -r requirements.txt
```

### Compiles and hot-reloads for development
```
python3 manage.py runserver
```

#### Инструменты
  - Django
  - DRF
  - AWS s3 хранилище
  - PostgreSQL
  - Heroku
